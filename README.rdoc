# Markup Editor

Markup Editor is a tiny sinatra application to render html pages from markup documents.

## Features

- Split screen view for editing and live preview
- Multiple markups: Markdown, RDoc, Textile
- Syntax highlighting
- Github-like styles
- Import gist documents
